개요

이 크롬 확장 프로그램은 MD, 퍼블리셔, 디자이너를 위한 도구 모음입니다. 특히 카페24 사이트 운영 시 유용한 기능들을 제공합니다.

크롬 확장 프로그램 사용 예시 (클릭하여 확대)

주요 기능

  • 모듈 표시: Cafe24 모듈 구조를 시각적으로 확인
  • 이미지 뱃지: 상품 이미지 파일명 표시 및 복사
  • 상품 번호 뱃지: 상품 번호 표시 및 복사
  • 스타일 인스펙터: 실시간 폰트 크기 및 스타일 조정

설치 방법

크롬 확장 프로그램을 설치하는 방법을 안내합니다.

1

확장 프로그램 파일 준비

확장 프로그램을 다운로드하거나 크롬 웹스토어에서 설치하세요.

파일 다운로드 웹스토어 설치
2

크롬 확장 프로그램 페이지 접속

크롬 브라우저에서 chrome://extensions/ 주소로 이동합니다. (클릭하여 복사)

3

개발자 모드 활성화

우측 상단의 "개발자 모드" 토글을 활성화합니다.

4

압축해제된 확장 프로그램 로드

"압축해제된 확장 프로그램을 로드" 버튼을 클릭하고 확장 프로그램 폴더를 선택합니다.

5

확장 프로그램 고정

확장 프로그램 목록에서 설치된 항목을 확인하고,
주소창 우측의 퍼즐 아이콘(🧩)을 클릭하여 고정해두면 더 편하게 사용할 수 있습니다.

🎉 설치 완료!

확장 프로그램이 성공적으로 설치되면 브라우저 툴바에 아이콘이 표시됩니다.

주요 기능

Cafe24 쇼핑몰 개발을 위한 핵심 기능들을 소개합니다.

1

모듈 표시 기능

Cafe24 쇼핑몰의 모듈 구조를 시각적으로 확인할 수 있는 기능입니다.

기능 설명
  • 모든 Cafe24 모듈에 보라색 테두리와 뱃지 표시
  • 모듈 클래스명을 뱃지에 표시
  • 뱃지 클릭 시 클래스명을 클립보드에 복사
사용 방법
  1. 확장 프로그램 아이콘 클릭
  2. "모듈 표시" 버튼 클릭
  3. 페이지의 모든 Cafe24 모듈이 하이라이트됨

⚠️ 주의사항: 이 기능은 Cafe24 쇼핑몰 사이트에서만 정상 작동합니다.

2

이미지 뱃지 기능

상품 이미지의 파일명을 표시하고 복사할 수 있는 기능입니다.

지원 형식
  • JPG, JPEG, PNG, WebP 등 일반 이미지
  • GIF 파일은 제외 (애니메이션 파일)
  • Lazy loading 이미지도 자동으로 로드
기능 특징
  • 자동 감지: 상품 상세 페이지의 모든 이미지
  • 파일명 추출: URL에서 실제 파일명 추출
  • 원클릭 복사: 뱃지 클릭으로 파일명 복사
3

상품 번호 뱃지

상품의 고유 번호를 표시하고 복사할 수 있는 기능입니다.

감지 방법
  • id="anchorBoxId_숫자" 속성에서 추출
  • class="anchorBoxId_숫자" 클래스에서 추출
  • 자동으로 모든 상품 요소를 스캔
표시 스타일
  • 보라색 점선 테두리
  • 좌측 상단 뱃지 위치
  • 보라색 배경, 흰색 텍스트
  • 복사 완료 시 체크 표시
4

스타일 인스펙터

실시간으로 요소의 스타일을 확인하고 조정할 수 있는 강력한 도구입니다. 실제 페이지 위에서 바로 조작되므로, 개발자 도구 없이도 확인 가능합니다.

주요 기능
  • 실시간 정보: 마우스 오버 시 요소 정보 표시
  • 폰트 크기 조절: + / - 키로 실시간 변경
  • 마진/패딩 조절: 키보드로 여백 조정
  • 화면 캡처: F2 키로 현재 상태 캡처
  • 텍스트 편집: 더블클릭으로 실시간 텍스트 수정
툴팁 정보
  • 요소 선택자, 크기, 폰트
  • 색상, 배경색 (HEX)
  • 패딩, 마진 값
  • 실시간 변경사항 표시

텍스트 편집 기능(25.07.28 심사 중)

스타일 인스펙터 모드에서 요소를 더블클릭하면 실시간으로 텍스트를 편집할 수 있습니다.

사용 방법
  1. 스타일 인스펙터 활성화
  2. 편집할 텍스트 요소에 더블클릭
  3. 텍스트 편집기에서 내용 수정
  4. Enter로 저장, ESC로 취소
주요 특징
  • 원본 텍스트 자동 백업
  • 실시간 스타일 적용
  • 빈 텍스트 방지
  • 편집 불가능한 요소 필터링

편집 중인 텍스트는 자동으로 원본 스타일이 적용되며, 0 키를 눌러 언제든지 원본으로 복원할 수 있습니다.

스타일 인스펙터 모드에서 사용할 수 있는 키보드 단축키입니다.

화면 캡처 F2

텍스트 편집 더블클릭

요소를 더블클릭하여 텍스트 편집

편집 저장 Enter

텍스트 편집 완료 및 저장

스타일 인스펙터 모드 종료

💡 팁

키보드 단축키는 스타일 인스펙터가 활성화된 상태에서만 작동합니다. 마우스로 요소위에 커서를 올려두고 단축키를 사용하세요.

활용 예시

크롬 확장 프로그램의 실제 활용 사례들을 소개합니다.

  • 페이지 시안과 실제 배치가 맞는지 확인할 때
  • 상품 이미지와 코드가 정확하게 들어갔는지 QA할 때
  • 특정 영역이 어떤 모듈인지 빠르게 파악할 때
  • 폰트 크기/간격 등을 웹 페이지에서 실시간으로 조정 테스트할 때

문제 해결

자주 발생하는 문제들과 해결 방법을 안내합니다.

일반적인 문제

1. 확장 프로그램이 작동하지 않음

  • 확장 프로그램이 활성화되어 있는지 확인
  • 페이지를 새로고침 후 다시 시도
  • 개발자 도구에서 오류 메시지 확인

2. 모듈이나 이미지가 표시되지 않음

  • 올바른 사이트에서 사용하고 있는지 확인
  • 페이지 로딩이 완료된 후 기능 활성화
  • 동적으로 로드되는 콘텐츠의 경우 페이지 스크롤 후 재시도

3. 스타일 인스펙터가 작동하지 않음

  • ESC 키를 눌러 이전 모드 종료 후 재시도
  • 페이지 새로고침 후 다시 활성화
  • 다른 확장 프로그램과의 충돌 확인

✨ 권장사항

  • 개발 작업 시에만 확장 프로그램 활성화
  • 정기적으로 브라우저 캐시 정리
  • 확장 프로그램 업데이트 확인

설치 방법

다운로드한 파일을 압축 해제하고 크롬에 설치하세요

  • 압축 해제: cafe24-module-badge.zip 파일을 압축 해제하여 폴더 생성
  • 크롬 확장 프로그램 페이지: 브라우저 주소창에 chrome://extensions/ 입력
  • 개발자 모드: 우측 상단의 "개발자 모드" 토글 활성화
  • 폴더 로드: "압축해제된 확장 프로그램을 로드" 버튼 클릭 후 폴더 선택
3

사용 방법

설치 완료 후 Cafe24 사이트에서 바로 사용하세요

  • 모듈 표시: 확장 프로그램 아이콘 클릭 시 모든 Cafe24 모듈에 보라색 테두리와 뱃지 표시
  • 이미지 뱃지: 상품 이미지에 청록색 테두리와 파일명 뱃지 표시
  • 상품 번호 뱃지: 상품 요소에 보라색 점선 테두리와 번호 뱃지 표시
  • 스타일 인스펙터: 마우스 오버로 요소 정보 확인 및 실시간 스타일 조정

✨ 이 확장 프로그램은 MD, 퍼블리셔, 디자이너를 위한 강력한 개발 도구입니다!
